JavaScript Objects

An object is an unordered collections of data, including primitive data, functions and other objects

Creating objects and accessing elements

Objects are composed from keys and properties

const user = { name: 'John'; age: 35 }

Creating objects and accessing elements

You can access objects elements using dot notation or square brackets

const user = { name: 'John'; age: 35 } console.log(user.name); // John console.log(user.age); // 35 const dynamicKey = 'age'; console.log(user['name']); // John console.log(user[dynamicKey]); // 35

Objects can be nested within other objects

const user = { name: 'John', contact: { phone: 123-123-123, address: { city: 'Baia Mare', country: 'Romania' } } } console.log(user.contact.address.city); // Baia Mare

What is this?

In JavaScript, the this keyword refers to an object.

Which object depends on how this is being invoked (used or called).

The this keyword refers to different objects depending on how it is used:


  1. In an object method, this refers to the object.
  2. Alone, this refers to the global object (window, environment).
  3. In a function, this refers to the global object.
  4. In an event, thishis refers to the element that received the event.

Object methods

Methods are actions can be performed on objects. Methods are functions stored as object properties.

const user = { firstName: 'Johnny', lastName: 'Deep', getFullName() { return `${this.firstName} ${this.lastName}` } } console.log(user.getFullName()); // Johnny Deep

Constructor and new operator

The {…} allow us to create a single object.

But when we want to create many similar object, we can use new operator. We can use constructor functions to make multiple similar objects.

function User(firstName,lastName){ this.firstName = firstName; this.lastName = lastName; } let user = new User('John', 34); let anotherUser = new User('Marry', 25);

Object useful methods

Object.keys() - retrieves the names of all the enumerable properties from an object. It is useful when you need to iterate over properties.

const user = { name: 'John'; age: 35 } console.log(Object.keys(user)); // ['name','age']

Object useful methods

Object.values() - it is like keys, but retrieve the values of an object. It's particularly useful when you are more interested in the data stored rather than their property names.

const user = { name: 'John'; age: 35 } console.log(Object.values(user)); // [ 'John', 35]

Getters and Setters

Getters and setters are special methods that provide access to object properties. Getters are used to read values of properties, while setters are used to write values to properties.

  1. Getters and setters allow you to control how your object properties are accessed and modified.
  2. Getters and setters can be used to validate data before it is set to an object.
  3. Getters and setters can be used to create properties that are calculated on the fly.

Get method

Getters are functions that are used to access properties on an object. They are similar to methods, but they are not called on an instance of an object. Instead, they are called on the object itself.

const user = { firstName: 'Johnny', lastName: 'Deep', get fullName() { return `${this.firstName} ${this.lastName}` } } console.log(user.fullName); // Johnny Deep

Set method

The setter is a method that allows you to set the value of a property in an object. It is a special type of method that is used to set the value of a property in an object. The setter method is a great way to validate data

let user = { name: 'John', set name(value) { if(value.length < 3) throw new Error('Name too short!') this._name = value; } };

JavaScript Arrays

Arrays store and organize multiple values in a flexible, indexed structure.

Creating Arrays and accessing elements

You can create arrays in JavaScript using square brackets [] and adding elements separated by commas.

let fruits = ["apple", "banana", "cherry"]; console.log(fruits[0]); // Output: "apple; console.log(fruits[2]); // Output: "cherry;

You can access elements in an array by their index, starting from 0.

Modifying Elements

You can change the value of an array element by assigning a new value to it.

let fruits = ["apple", "banana", "cherry"]; console.log(fruits); // Output: ["apple", kiwi", "cherry"];

Adding and Removing Elements

You can add elements to the end of an array using the push method and remove elements from the end using the pop method.

let fruits = ["apple", "banana", "cherry"]; fruits.push("orange"); // Output: ["apple", "kiwi", "cherry", "orange"] fruits.pop(); // Output: ["apple", "kiwi", "cherry"];

Iterating through Arrays:

You can use loops to iterate through array elements.

let fruits = ["apple", "banana", "cherry"]; for (let i = 0; i fruits.length; i++) { console.log(fruits[i]); }; for (const fruit of fruits) { console.log(fruit); }

Iterating through Arrays:

The forEach() method of Array instances executes a provided function once for each array element.

let fruits = ["apple", "banana", "cherry"]; fruits.forEach((element) => { console.log(element.toUpperCase()); }); // APPLE, BANANA, CHERRY

Changing or removing arrays elements

The splice() method of Array instances changes the contents of an array by removing or replacing existing elements and/or adding new elements in place.

Delete an element:

let fruits = ["apple", "banana", "cherry"]; fruits.splice(1, 1) // from index 1 remove one element // apple, cherry

Changing or removing arrays elements

Delete multiple elements with splice():

let fruits = ["apple", "banana", "cherry"]; fruits.splice(0, 2) // from index 0 remove two elements // cherry

Changing or removing arrays elements

Insert elements using splice():

For that we need to set deleteCount to 0:

let fruits = ["apple", "banana", "cherry"]; fruits.splice(1, 0, "strawberry", "orange") // from index 1 // insert two elements // apple, banana, cherry, strawberry, orange

Changing or removing arrays elements

Modifying elements with slice():

It returns a new array copying to it all items from index start to end (not including end). The original array will not be modified.

let fruits = ["apple", "banana", "cherry"]; fruits.splice(1, 3) // returns elements from 1 to 3 // banana, cherry

Searching in arrays

Using includes():

The includes() method returns either a true or a false if a value exists in an array or not.

let fruits = ["apple", "banana", "cherry"]; fruits.includes("apple") // true fruits.includes("lemon") // false

Searching in arrays

Using indexOf():

The indexOf() method returns the first index of a value in an array. If there is no match, the method returns -1.

let fruits = ["apple", "banana", "cherry"]; fruits.indexOf("apple") // 0 fruits.includes("lemon") // -1

Searching in arrays

Using find():

The find() method returns the first value in an array that matches the conditions of a function. If there is no match, the method returns undefined.

let fruits = ["apple", "banana", "cherry"]; fruits.find(element => element.length === 6) // banana

Searching in arrays

Using filter():

The filter() method returns a new array of all the values in an array that matches the conditions of a function. If there is no match, the method returns an empty array.

let fruits = ["apple", "banana", "cherry"]; fruits.filter(element => element.length === 6) // banana, cherry

Transform arrays

map()

It calls the callback function for each element of the array and returns the array of results.

let fruits = ["apple", "banana", "cherry"]; fruits.map(element => element.length) // 5, 6, 6

Transform arrays

reduce()

This method reduces an array of values down to just one value. The single value that is returned can be of any type.

let fruits = ["apple", "banana", "cherry"]; fruits.reduce((total, current) => total + ', ' + current) // 'apple, banana, cherry'

Transform arrays

reduce()
let arr = [1, 2, 3]; arr.reduce((total, current) => total + current) // 6
